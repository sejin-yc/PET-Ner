## 📝 참고사항

1. **Pi Gateway의 역할**: 
   - 하드웨어 제어 (STM32 통신)
   - 통신 브릿지 (Web/BE/AI ↔ STM32)
   - 안전 제어 (비상 정지, 속도 감소)
   - 기본 인식 (고양이 탐지)

2. **타 시스템 담당**:
   - **젯슨 AI**: 사료 잔량 인식, 배변 상태 인식, 장기간 행동 분석
   - **BE**: 알림 발송, 로그 저장, 리포트 생성
   - **Nav2**: 자율 주행 경로 계획, 충전 스테이션 도킹
   - **별도 서버**: 영상 스트리밍 (RTSP/WebRTC)

3. **추가 구현 가능**:
   - TTS: STM32에 UART 프로토콜 확장 필요
   - 패트롤 액션 스케줄 UI: FE에서 추가 가능 (API는 이미 구현됨)
