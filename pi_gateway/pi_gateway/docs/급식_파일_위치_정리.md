# 급식 관련 파일 위치 정리

## 📁 파일 위치

### Pi Gateway 폴더 (`pi_gateway/`)
**이미 구현 완료된 파일들:**
- ✅ `src/ros_cmdvel.py` - `FeedControlBridge` 클래스 (급식 요청/수신)
- ✅ `src/main.py` - `FeedControlBridge` 생성 및 연결
- ✅ `docs/급식_젯슨_연동_가이드.md` - 젯슨 담당자용 가이드
- ✅ `docs/급식_로직_전체_흐름.md` - 전체 흐름 설명

**Pi Gateway가 하는 일:**
- `feed/request` 토픽 발행 (젯슨에 요청)
- `feed/amount` 토픽 구독 (젯슨이 계산한 결과 받음)
- UART로 STM32에 전송

---

### 젯슨 폴더 (`S14P11C203-Feed_AI/`)
**젯슨에서 실행되어야 하는 파일들:**
- `src/infer/stream_server_ros.py` - ROS2 이미지 구독 및 추론
- `src/infer/infer_cam_rt.py` - 카메라 직접 연결 버전
- `models/bowl_seg_best.pt` - YOLO 세그멘테이션 모델
- `models/ckpt_feedg_roi_fromtxt__effv2s__224__e80__bs32__lr3e-4__seed42.pt` - Regression 모델
- `models/config.yaml` - 설정 파일 (full_grams: 427.8)

**젯슨이 하는 일:**
- `feed/request` 토픽 구독 (Pi Gateway 요청 받음)
- FEED_AI 모델로 사료량 계산
- `feed/amount` 토픽 발행 (계산 결과 보냄)
