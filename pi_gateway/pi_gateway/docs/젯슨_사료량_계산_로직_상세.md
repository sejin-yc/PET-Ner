# 젯슨 사료량 계산 로직 상세

Feed_AI 폴더의 코드를 분석한 결과입니다.

---

## 📋 전체 프로세스

```
1. 카메라 이미지 받기 (ROS2 토픽: /camera/image_raw)
   ↓
2. YOLO 세그멘테이션으로 그릇 찾기
   ↓
3. 그릇 ROI(Region of Interest) 추출
   ↓
4. Regression 모델로 사료 량 계산 (그램 단위)
   ↓
5. 퍼센트 계산 (grams / full_grams * 100)
   ↓
6. 레벨 변환 (1~3)
```

---

## 🔍 단계별 상세 설명

### 1단계: 이미지 받기

```python
# ROS2 토픽에서 이미지 구독
frame, ts = sub.get_latest()  # BGR 형식
```

- 토픽: `/camera/image_raw` (sensor_msgs/Image)
- 형식: BGR (OpenCV 기본)

---

### 2단계: YOLO 세그멘테이션으로 그릇 찾기

```python
# YOLO 모델 로드
yolo = YOLO("models/bowl_seg_best.pt")

# 세그멘테이션 실행
r = yolo.predict(
    source=frame, 
    imgsz=640,      # 이미지 크기
    conf=0.60,      # 신뢰도 임계값
    verbose=False
)[0]

# 가장 좋은 그릇 인스턴스 선택
idx, reason = choose_best_instance_by_mask_area(
    r, 
    conf_thres=0.60, 
    min_mask_area_ratio=0.02  # 최소 마스크 영역 비율
)
```

**선택 기준:**
- 신뢰도(confidence)가 0.60 이상
- 마스크 영역이 전체 프레임의 2% 이상
- 마스크 영역이 가장 큰 인스턴스 선택

**모델 파일:** `models/bowl_seg_best.pt` (YOLOv8 세그멘테이션)

---

### 3단계: 그릇 ROI 추출

```python
# 세그멘테이션 결과에서 ROI 추출
roi, bbox = get_roi_from_seg_result(r, frame, idx, margin=0.12)

# margin=0.12: 바운딩 박스를 12% 확장 (여유 공간)
```

**과정:**
1. 마스크에서 바운딩 박스 계산
2. 바운딩 박스를 12% 확장 (여유 공간)
3. ROI 영역 추출 (BGR 이미지)

**ROI (Region of Interest):** 그릇 부분만 잘라낸 이미지

---

### 4단계: Regression 모델로 사료 량 계산 (그램)

```python
# Regression 모델 로드
reg_model, reg_tf, backbone, reg_imgsz = load_regression(
    Path("models/ckpt_feedg_roi_fromtxt__effv2s__224__e80__bs32__lr3e-4__seed42.pt"),
    device="cuda"
)

# 사료 량 계산 (그램 단위)
grams = predict_grams(reg_model, reg_tf, roi, device="cuda", amp=True)
```

**모델 정보:**
- 백본: EfficientNetV2-S (`effv2s`)
- 입력 크기: 224x224
- 출력: 사료 량 (그램, float)

**예측 과정:**
```python
@torch.no_grad()
def predict_grams(model, tf, roi_bgr, device: str, amp: bool) -> float:
    # 1. BGR → RGB 변환
    rgb = cv2.cvtColor(roi_bgr, cv2.COLOR_BGR2RGB)
    
    # 2. PIL Image로 변환
    x = tf(Image.fromarray(rgb))  # Resize + ToTensor
    
    # 3. 배치 차원 추가
    x = x.unsqueeze(0).to(device)
    
    # 4. 모델 추론 (AMP 사용 가능)
    if amp and device.startswith("cuda"):
        with torch.autocast(device_type="cuda", dtype=torch.float16):
            g = float(model(x).item())
    else:
        g = float(model(x).item())
    
    return g  # 그램 단위
```

**모델 파일:** `models/ckpt_feedg_roi_fromtxt__effv2s__224__e80__bs32__lr3e-4__seed42.pt`

---

### 5단계: 퍼센트 계산

```python
# config.yaml에서 full_grams 읽기
full_grams = 427.8  # 그릇 + 사료 기준 "가득 찼을 때" 무게

# 퍼센트 계산
pct = (grams / full_grams) * 100.0
pct = clamp(pct, 0.0, 100.0)  # 0~100%로 제한
```

**설정 파일:** `models/config.yaml`
```yaml
full_grams: 427.8  # 가득 찼을 때 무게 (그램)
```

**예시:**
- `grams = 200.0g` → `pct = (200.0 / 427.8) * 100 = 46.7%`
- `grams = 427.8g` → `pct = 100%` (가득 참)

---

### 6단계: 레벨 변환 (1~3)

**Pi Gateway가 받을 값:**

```python
# 퍼센트를 레벨로 변환
if pct < 33.0:
    feed_level = 1  # 적음 (0~33%)
elif pct < 66.0:
    feed_level = 2  # 보통 (33~66%)
else:
    feed_level = 3  # 많음 (66~100%)
```

**레벨 의미:**
- **레벨 1**: 사료가 적음 (0~33%)
- **레벨 2**: 사료가 보통 (33~66%)
- **레벨 3**: 사료가 많음 (66~100%)

## 📊 전체 코드 흐름 (요약)

```python
# 1. 모델 로드
yolo = YOLO("models/bowl_seg_best.pt")
reg_model, reg_tf, _, _ = load_regression(
    Path("models/ckpt_feedg_roi_fromtxt__effv2s__224__e80__bs32__lr3e-4__seed42.pt"),
    device="cuda"
)
full_grams = 427.8  # config.yaml에서 읽기

# 2. 이미지 받기
frame = get_latest_image()  # ROS2 토픽에서

# 3. 그릇 찾기 (YOLO 세그멘테이션)
result = yolo.predict(source=frame, imgsz=640, conf=0.60)[0]
idx, reason = choose_best_instance_by_mask_area(result, conf_thres=0.60)

# 4. ROI 추출
roi, bbox = get_roi_from_seg_result(result, frame, idx, margin=0.12)

# 5. 사료 량 계산 (그램)
grams = predict_grams(reg_model, reg_tf, roi, device="cuda", amp=True)

# 6. 퍼센트 계산
percent = (grams / full_grams) * 100.0
percent = clamp(percent, 0.0, 100.0)

# 7. 레벨 변환
if percent < 33.0:
    feed_level = 1
elif percent < 66.0:
    feed_level = 2
else:
    feed_level = 3

# 8. feed/amount 토픽 발행
pub_feed_amount.publish(feed_level)  # std_msgs/UInt8
```

## 🔧 젯슨 담당자가 추가해야 할 코드

`stream_server_ros.py`에 추가:

```python
from std_msgs.msg import String, UInt8

class FeedAINode(Node):
    def __init__(self):
        super().__init__("feed_ai_node")
        
        # feed/request 구독
        self.sub_feed_request = self.create_subscription(
            String, "feed/request", self.on_feed_request, 10
        )
        
        # feed/amount 발행
        self.pub_feed_amount = self.create_publisher(UInt8, "feed/amount", 10)
        
        # 기존 코드...
        self.yolo = YOLO("models/bowl_seg_best.pt")
        self.reg_model, self.reg_tf, _, _ = load_regression(...)
        self.full_grams = 427.8
    
    def on_feed_request(self, msg: String):
        """Pi Gateway가 급식 요청 시 호출"""
        # 즉시 추론 실행
        self.process_feed_request()
    
    def process_feed_request(self):
        """사료 량 계산 및 feed/amount 발행"""
        # 1. 이미지 받기
        frame, ts = self.sub_image.get_latest()
        if frame is None:
            return
        
        # 2. YOLO 세그멘테이션
        r = self.yolo.predict(source=frame, imgsz=640, conf=0.60)[0]
        idx, reason = choose_best_instance_by_mask_area(r, conf_thres=0.60)
        
        if idx is None:
            feed_level = 2  # 기본값
        else:
            # 3. ROI 추출
            roi, bbox = get_roi_from_seg_result(r, frame, idx, margin=0.12)
            if roi is None or roi.size == 0:
                feed_level = 2  # 기본값
            else:
                # 4. 사료 량 계산 (그램)
                grams = predict_grams(
                    self.reg_model, 
                    self.reg_tf, 
                    roi, 
                    device="cuda", 
                    amp=True
                )
                
                # 5. 퍼센트 계산
                percent = (grams / self.full_grams) * 100.0
                percent = clamp(percent, 0.0, 100.0)
                
                # 6. 레벨 변환
                if percent < 33.0:
                    feed_level = 1
                elif percent < 66.0:
                    feed_level = 2
                else:
                    feed_level = 3
        
        # 7. feed/amount 발행
        msg = UInt8()
        msg.data = feed_level
        self.pub_feed_amount.publish(msg)
```
