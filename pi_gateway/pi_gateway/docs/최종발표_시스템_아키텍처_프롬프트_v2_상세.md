### 통신 경로 (모든 화살표에 프로토콜 라벨 필수)

**화살표 스타일 및 라벨:**
- MQTT: 점선 초록색 화살표, 라벨 "MQTT" 필수
- HTTP REST: 실선 파란색 화살표, 라벨 "HTTP REST" 필수
- WebSocket STOMP: 실선 보라색 화살표, 라벨 "WebSocket STOMP" 필수
- WebRTC P2P: 실선 주황색 화살표, 라벨 "WebRTC" 필수
- ROS2 DDS: 실선 초록색 화살표, 라벨 "ROS2 DDS" 필수
- UART Serial: 실선 주황색 화살표, 라벨 "UART" 필수

### 필수 연결 경로 (모두 화살표로 표시)

#### 1. FE → BE 직접 연결 (반드시 포함)
- **HTTP REST**: FE (Axios) → Nginx (EC2) → BE (Spring Boot) 
  - 라벨: "HTTP REST"
  - 용도: 인증 (/api/user/login, /api/user/register), 고양이 관리 (/api/cats), 영상 갤러리 (/api/videos), 로그 (/api/logs), TTS (/api/robot/tts)
- **WebSocket STOMP**: FE → Nginx (EC2) → BE (Spring Boot WebSocket STOMP 브로커)
  - 라벨: "WebSocket STOMP"
  - 용도: WebRTC 시그널링 (/ws, /sub/peer/offer, /pub/peer/answer)

#### 2. FE → Infra (MQTT)
- **MQTT WebSocket**: FE (Paho MQTT.js) → Nginx (/mqtt) → Mosquitto (EC2)
  - 라벨: "MQTT"
  - 용도: 수동 조작 (robot/{userId}/control 발행), 로봇 상태 수신 (robot/{userId}/status 구독)

#### 3. BE → Infra (MQTT)
- **MQTT TCP**: BE (MqttService) → Mosquitto (EC2, 1883)
  - 라벨: "MQTT"
  - 용도: TTS 명령 발행 (robot/{userId}/control), cat_state 구독 (/#)

#### 4. BE → PostgreSQL (EC2 내부)
- **JDBC**: BE (Spring Boot) → PostgreSQL (EC2 내부, 5432)
  - 라벨: "JDBC"
  - 용도: 데이터 저장 (User, Cat, Video, Log, Notification)

#### 5. Infra → CORE/GATEWAY
- **MQTT TCP**: Mosquitto (EC2) → Pi Gateway (MQTT Pi Bridge)
  - 라벨: "MQTT"
  - 용도: TTS 명령 전달, cat_state 수신

#### 6. FE → CORE/GATEWAY (직접, 선택적)
- **HTTP REST**: FE (Axios) → Pi Gateway (REST API, /control, /action, /telemetry)
  - 라벨: "HTTP REST"
  - 용도: 직접 제어 (선택적, 주로 MQTT 사용)

#### 7. CORE/GATEWAY → ROS2
- **ROS2 DDS**: Pi Gateway → ROS2 노드들
  - 라벨: "ROS2 DDS"
  - 용도: /cmd_vel_joy 발행, /cmd_vel_auto 구독, /odom 발행, /telemetry/* 발행, /patrol/* 토픽

#### 8. ROS2 → CORE/GATEWAY
- **ROS2 DDS**: ROS2 노드들 → Pi Gateway
  - 라벨: "ROS2 DDS"
  - 용도: /cmd_vel_auto 발행, /patrol/waypoint_reached 발행

#### 9. CORE/GATEWAY → AI (Jetson)
- **ROS2 DDS**: Pi Gateway → Jetson
  - 라벨: "ROS2 DDS"
  - 용도: /feed/request 발행, /arm/start, /arm/water 구독

#### 10. AI (Jetson) → CORE/GATEWAY
- **ROS2 DDS**: Jetson → Pi Gateway
  - 라벨: "ROS2 DDS"
  - 용도: /feed/amount 발행, /arm/job_complete 발행

#### 11. ROS2 → AI (Jetson)
- **ROS2 DDS**: ROS2 카메라 퍼블리셔 → Jetson
  - 라벨: "ROS2 DDS"
  - 용도: /front_cam/compressed 구독 (고양이 탐지, WebRTC)

#### 12. AI (Jetson) → ROS2
- **ROS2 DDS**: Jetson → ROS2
  - 라벨: "ROS2 DDS"
  - 용도: /arm/start, /arm/water, /arm/position_correct 발행

#### 13. CORE/GATEWAY → HW (STM32)
- **UART Serial**: Pi Gateway → STM32
  - 라벨: "UART"
  - 용도: CMD_VEL (0x01), FEED (0x05), ARM_START (0x06), ESTOP (0x10) 등

#### 14. HW (STM32) → CORE/GATEWAY
- **UART Serial**: STM32 → Pi Gateway
  - 라벨: "UART"
  - 용도: BATTERY (0x81), ENCODER (0x82), IMU (0x83), STATUS (0x84)

#### 15. ROS2 → HW (STM32) - 직접 연결 없음
- **참고**: ROS2는 Pi Gateway를 통해 STM32와 통신 (직접 연결 없음)

#### 16. FE → 로봇 (WebRTC 스트리밍)
- **WebSocket STOMP**: FE → BE (STOMP 브로커) → 로봇 (robot_webrtc.py)
  - 라벨: "WebSocket STOMP"
  - 용도: Offer/Answer 시그널링
- **WebRTC P2P**: FE ↔ 로봇 (robot_webrtc.py)
  - 라벨: "WebRTC"
  - 용도: 영상 스트리밍 (P2P, STUN 서버 사용)

#### 17. 로봇 → BE (고양이 탐지)
- **MQTT**: cat_detection_service.py → Mosquitto (EC2) → BE (MqttService)
  - 라벨: "MQTT"
  - 용도: robot/{userId}/cat_state 발행
- **HTTP REST**: cat_detection_service.py → Nginx (EC2) → BE (Spring Boot)
  - 라벨: "HTTP REST"
  - 용도: POST /api/videos (영상 메타 저장)
