# 파일 동기화 가이드 (로봇으로 전송)

수정한 파일을 라즈베리파이로 빠르게 옮기는 방법입니다.

---

## 🚀 빠른 방법 (추천)

### 방법 1: rsync 사용 (변경된 파일만 자동 전송)

```bash
cd /home/ssafy/Downloads/pi_gateway_spec_aligned/pi_gateway

# 스크립트 사용 (가장 편함)
./scripts/sync_to_robot.sh

# 또는 직접 실행
rsync -av --delete \
  --exclude='venv/' \
  --exclude='__pycache__/' \
  --exclude='*.pyc' \
  --exclude='.git/' \
  ./ \
  c203@192.168.100.254:~/pi_gateway/
```

**장점:**
- 변경된 파일만 자동으로 전송
- 빠르고 효율적
- venv, 캐시 파일 제외

---

### 방법 2: 특정 파일만 전송

```bash
cd /home/ssafy/Downloads/pi_gateway_spec_aligned/pi_gateway

# 스크립트 사용
./scripts/quick_sync.sh

# 또는 직접 실행
scp src/ros_telemetry_bridge.py c203@192.168.100.254:~/pi_gateway/src/
scp src/main.py c203@192.168.100.254:~/pi_gateway/src/
scp config/params.yaml c203@192.168.100.254:~/pi_gateway/config/
```

**장점:**
- 필요한 파일만 선택적으로 전송
- 매우 빠름

---

### 방법 3: 압축 파일 (큰 변경 시)

```bash
cd /home/ssafy/Downloads/pi_gateway_spec_aligned
tar -czf pi_gateway.tgz pi_gateway
scp pi_gateway.tgz c203@192.168.100.254:~/

# 로봇에서
ssh c203@192.168.100.254
tar -xzf ~/pi_gateway.tgz
```

**장점:**
- 전체 폴더를 한 번에 전송
- 초기 배포나 큰 변경 시 유용

---

## ⚙️ 스크립트 설정

### sync_to_robot.sh 설정

스크립트 파일을 열어서 로봇 정보 수정:

```bash
nano scripts/sync_to_robot.sh
```

**수정:**
```bash
ROBOT_USER="c203"        # 로봇 계정
ROBOT_IP="192.168.100.254"  # 로봇 IP
ROBOT_PATH="~/pi_gateway"   # 로봇 경로
```

## 🔧 문제 해결

### rsync가 없을 때

```bash
# 설치
sudo apt install rsync

# 또는 scp 사용
scp -r src/ c203@192.168.100.254:~/pi_gateway/
```

### 권한 문제

```bash
# SSH 키 설정 (비밀번호 없이 접속)
ssh-copy-id c203@192.168.100.254
```

### 네트워크 문제

```bash
# 연결 확인
ping 192.168.100.254

# SSH 연결 확인
ssh c203@192.168.100.254
```
