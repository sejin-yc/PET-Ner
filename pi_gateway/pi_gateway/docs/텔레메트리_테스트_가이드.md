# 텔레메트리 테스트 가이드

Pi Gateway에서 실시간 텔레메트리 데이터를 받아보는 방법입니다.

---

## 📡 텔레메트리 WebSocket

Pi Gateway는 `/ws/telemetry` 엔드포인트를 통해 실시간 텔레메트리 데이터를 전송합니다.

- **엔드포인트**: `ws://localhost:8000/ws/telemetry`
- **전송 주기**: 0.2초 (200ms)마다 자동 전송
- **데이터 형식**: JSON

**방법 B: 브라우저 콘솔에서 직접**

```javascript
// F12 → Console 탭에서 실행
const ws = new WebSocket('ws://localhost:8000/ws/telemetry');

ws.onopen = () => {
  console.log('✅ 텔레메트리 연결 성공!');
};

ws.onmessage = (e) => {
  const data = JSON.parse(e.data);
  console.log('📩 텔레메트리:', data);
  
  // 배터리 정보
  if (data.battery) {
    console.log('🔋 배터리:', {
      soc: data.battery.soc_percent + '%',
      voltage: (data.battery.vbat_mV / 1000) + 'V',
      charging: data.battery.charging
    });
  }
  
  // IMU 정보
  if (data.imu) {
    console.log('🧭 IMU:', {
      yaw: data.imu.yaw + '°',
      roll: data.imu.roll + '°',
      pitch: data.imu.pitch + '°'
    });
  }
  
  // 엔코더 정보
  if (data.encoders) {
    console.log('⚙️ 엔코더:', {
      left: data.encoders.left,
      right: data.encoders.right,
      velocity: data.encoders.velocity + ' m/s'
    });
  }
};

ws.onerror = (e) => {
  console.error('❌ 오류:', e);
};

ws.onclose = () => {
  console.log('🔌 연결 종료');
};
```

## 📊 텔레메트리 데이터 구조

### 전체 구조

```json
{
  "type": "telemetry",
  "ts": 1234567890.123,
  "battery": {
    "soc_percent": 85,
    "vbat_mV": 11600,
    "charging": false
  },
  "imu": {
    "roll": 0.5,
    "pitch": -0.2,
    "yaw": 1.57,
    "accel_x": 0.0,
    "accel_y": 0.0,
    "accel_z": 9.8
  },
  "encoders": {
    "left": 1234,
    "right": 1235,
    "velocity": 0.5
  }
}
```

### 필드 설명

#### `battery` (배터리 정보)
- `soc_percent`: 배터리 잔량 (%)
- `vbat_mV`: 배터리 전압 (mV)
- `charging`: 충전 중 여부 (boolean)

#### `imu` (IMU 센서 정보)
- `roll`: 롤 각도 (도)
- `pitch`: 피치 각도 (도)
- `yaw`: 요 각도 (도)
- `accel_x`, `accel_y`, `accel_z`: 가속도 (m/s²)

#### `encoders` (엔코더 정보)
- `left`: 왼쪽 엔코더 값
- `right`: 오른쪽 엔코더 값
- `velocity`: 현재 속도 (m/s)

## 🔍 데이터 소스

텔레메트리 데이터는 다음 소스에서 수집됩니다:

1. **ROS2 Topics** (ROS2가 활성화된 경우):
   - `telemetry/battery` → `battery` 필드
   - `telemetry/imu` → `imu` 필드
   - `telemetry/encoders` → `encoders` 필드

2. **UART** (실기기 연결 시):
   - STM32에서 직접 수신한 데이터

3. **데모 모드**:
   - ROS2가 없으면 대부분 `null` 또는 기본값
   - ROS2가 활성화되어 있으면 ROS2 퍼블리셔에서 데이터 수신

## 💡 팁

1. **데이터가 없을 때:**
   - ROS2가 실행 중인지 확인
   - `ros2 topic list`로 텔레메트리 토픽 확인
   - `ros2 topic echo /telemetry/battery`로 직접 확인

2. **연결이 안 될 때:**
   - Pi Gateway가 실행 중인지 확인
   - `http://localhost:8000/docs` 접속 가능한지 확인
   - 브라우저 콘솔에서 WebSocket 오류 확인

3. **데이터 업데이트가 안 될 때:**
   - Pi Gateway 콘솔에서 ROS2 구독자 로그 확인
   - 텔레메트리 퍼블리셔가 실행 중인지 확인

## 🔧 고급: Python 스크립트로 테스트

```python
#!/usr/bin/env python3
import asyncio
import websockets
import json

async def test_telemetry():
    uri = "ws://localhost:8000/ws/telemetry"
    async with websockets.connect(uri) as websocket:
        print("✅ 텔레메트리 연결 성공!")
        
        while True:
            try:
                data = await websocket.recv()
                telemetry = json.loads(data)
                
                print("\n" + "="*50)
                print(f"타임스탬프: {telemetry.get('ts')}")
                
                if telemetry.get('battery'):
                    batt = telemetry['battery']
                    print(f"🔋 배터리: {batt.get('soc_percent', 'N/A')}% "
                          f"({batt.get('vbat_mV', 0)/1000:.2f}V) "
                          f"{'충전 중' if batt.get('charging') else '방전 중'}")
                
                if telemetry.get('imu'):
                    imu = telemetry['imu']
                    print(f"🧭 IMU: Yaw={imu.get('yaw', 0):.2f}° "
                          f"Roll={imu.get('roll', 0):.2f}° "
                          f"Pitch={imu.get('pitch', 0):.2f}°")
                
                if telemetry.get('encoders'):
                    enc = telemetry['encoders']
                    print(f"⚙️ 엔코더: L={enc.get('left', 0)} "
                          f"R={enc.get('right', 0)} "
                          f"속도={enc.get('velocity', 0):.2f}m/s")
                
            except Exception as e:
                print(f"❌ 오류: {e}")
                break

if __name__ == "__main__":
    asyncio.run(test_telemetry())
```

**사용 방법:**
```bash
# websockets 라이브러리 설치
pip install websockets

# 스크립트 실행
python3 test_telemetry.py
```

## 📝 참고

- 텔레메트리 WebSocket은 **서버 푸시 방식**입니다 (클라이언트가 요청할 필요 없음)
- 0.2초마다 자동으로 최신 데이터를 전송합니다
- 클라이언트는 연결만 하면 자동으로 데이터를 받을 수 있습니다
- 여러 클라이언트가 동시에 연결 가능합니다
