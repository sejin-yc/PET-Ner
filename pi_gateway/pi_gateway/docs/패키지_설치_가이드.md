# 패키지 설치 가이드

Pi Gateway 실행에 필요한 Python 패키지를 설치하는 방법입니다.

---

## 🚀 빠른 설치

### 방법 1: requirements.txt 사용 (권장)

```bash
cd ~/pi_gateway

# 가상환경 사용 (권장)
python3 -m venv venv
source venv/bin/activate

# 패키지 설치
pip install -r requirements.txt
```

### 방법 2: 직접 설치

```bash
pip install fastapi uvicorn pyyaml pyserial paho-mqtt requests websocket-client numpy
```

---

## 📋 필요한 패키지

- `fastapi` - 웹 서버 프레임워크
- `uvicorn` - ASGI 서버
- `pyyaml` - YAML 설정 파일 파싱
- `pyserial` - UART 통신
- `paho-mqtt` - MQTT 통신 (선택)
- `requests` - HTTP 요청 (선택)
- `websocket-client` - WebSocket 클라이언트 (선택)
- `numpy` - 수치 계산 (ROS2 사용 시)

---

## ⚠️ 문제 해결

### pip가 없을 때

```bash
sudo apt update
sudo apt install python3-pip
```

### 권한 오류

```bash
# 가상환경 사용 (권장)
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 또는 사용자 설치
pip install --user -r requirements.txt
```

### ROS2 패키지 오류

ROS2를 사용하지 않으면 무시해도 됩니다. 데모 모드에서는 ROS2가 필요 없습니다.

## 💡 가상환경 사용 (권장)

```bash
cd ~/pi_gateway

# 가상환경 생성
python3 -m venv venv

# 가상환경 활성화
source venv/bin/activate

# 패키지 설치
pip install -r requirements.txt

# 실행
python3 src/main.py

# 가상환경 비활성화 (나중에)
deactivate
```
